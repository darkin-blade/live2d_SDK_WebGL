## 使用(只介绍最新版的使用方式)

> 需要`php`环境

- 一键加载(sample.html)

在`JSManager.js`中调整`minNum`和`maxNum`的大小即可

- 修改样式

`live2d.css`和`tips.css`以及`LAppDefine.js`中的大小参数

- 修改模型(源代码不含模型资源)

`LAppDefine.js`中自行修改
